# 重启仲裁服务

本节为您介绍如何重启仲裁服务。

## 前提条件

* 仲裁服务需处于 **STOPPED**、**RUNNING** 或 **UNAVAILABLE** 状态。
* 仲裁服务关联的租户中未存在日志流降级。

## 操作步骤

1. 登录 OCP 。

2. 在左导航栏单击 **集群** ，进入集群页面。

3. 在总览页面的右上角，单击 **...** 图标，选择 **仲裁服务管理** 。

4. 在仲裁服务列表页面，选择需要重启的仲裁服务，将光标悬停于右侧操作列的 **...**，在选择框中单击 **重启服务**。

5. 在弹窗中单击 **启动**。

    <main id="notice" type='notice'>
    <h4>注意</h4>
    <p>重启过程中，仲裁服务所关联的集群和租户将无法使用该仲裁服务，请谨慎操作。</p>
    </main>
