# 启动仲裁服务

本节为您介绍如何启动仲裁服务。当仲裁服务处于停止状态时，您可通过该功能启动仲裁服务。

<main id="notice" type='notice'>
<h4>功能适用性</h4>
<p>该内容仅适用于 OCP 企业版，OCP 社区版不支持此功能。</p>
</main>

## 前提条件

仲裁服务处于 **STOPPED** 状态。

## 操作步骤

1. 登录 OCP 。

2. 在左导航栏单击 **集群** ，进入集群页面。

3. 在总览页面的右上角，单击 **...** 图标，选择 **仲裁服务管理** 。

4. 在仲裁服务列表页面，选择需要启动的仲裁服务，单击右侧操作列的 **启动服务**。

5. 在弹窗中单击 **启动**。

    启动服务后，已关联的集群和租户将继续使用该仲裁服务。
