# 停止仲裁服务

当您不再需要使用仲裁服务时，您可通过该功能进行停止。

## 前提条件

* 仲裁服务处于 **RUNNING** 状态。
* 仲裁服务关联的租户中未存在日志流降级。

## 操作步骤

1. 登录 OCP 。

2. 在左导航栏单击 **集群** ，进入集群页面。

3. 在总览页面的右上角，单击 **...** 图标，选择 **仲裁服务管理** 。

4. 在仲裁服务列表页面，选择需要停止的仲裁服务，单击右侧操作列的 **停止服务**。

5. 在弹窗中单击 **停止**。

    <main id="notice" type='notice'>
    <h4>注意</h4>
    <p>停止仲裁服务后，仲裁服务所关联的集群和租户将无法使用该仲裁服务，请谨慎操作。</p>
    </main>
