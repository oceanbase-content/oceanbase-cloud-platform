# 关闭死锁自动检测

您可根据您的业务需求，关闭死锁自动检测。

## 前提条件

* 当前登录账号具备集群的管理权限。

* 当前集群版本为 V3.2.0 及以上。

* 集群已开启死锁自动检测功能。

## 操作步骤

1. 登录 OCP 。

2. 在左侧导航栏单击 **集群** ，系统默认进入 **集群列表** 页签。

3. 在 **集群列表** 中选择待操作的集群并单击其集群名。

4. 在集群 **概览** 页面，单击右上角 **...** 图标，选择 **关闭死锁自动检测** 。

5. 在对话框中单击 **确定** ，即可关闭死锁自动检测功能。
