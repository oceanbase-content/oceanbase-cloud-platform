# 采集租户临时文件空间占用

## 采集描述

查看租户在节点上产生的临时文件大小。

## 采集原理

```sql
SELECT svr_ip, svr_port, file_type, round(sum(data_size) / 1024 / 1024 / 1024, 3) AS size_gb
FROM __all_space_usage
WHERE file_type = 'tenant tmp data'
GROUP BY svr_ip, svr_port;
```
