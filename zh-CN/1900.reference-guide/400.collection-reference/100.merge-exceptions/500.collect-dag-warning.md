# 采集 OB 内核 DAG 警告信息

## 采集描述

针对转储/合并失败的情况，查询 `__all_virtual_dag_warning_history` 虚拟表获取失败原因及 trace ID。

## 采集原理

3.2.0 及后版本：

```sql
SELECT * FROM __all_virtual_dag_warning_history;
```
