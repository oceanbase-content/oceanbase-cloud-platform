# QPS

本节为您介绍 OBProxy 的监控指标 **QPS** 。

## commit

### 指标介绍

OBProxy 集群平均每秒转发的 commit 请求数。

### 指标参数说明

| **指标项** |       **指标名称**       | **单位** |
|---------|----------------------|--------|
| commit  | request_total_commit | 个      |

### 计算表达式

sum(rate(odp_sql_request_total{sqltype="COMMIT", @LABELS}[@INTERVAL])) by (@GBLABELS)
