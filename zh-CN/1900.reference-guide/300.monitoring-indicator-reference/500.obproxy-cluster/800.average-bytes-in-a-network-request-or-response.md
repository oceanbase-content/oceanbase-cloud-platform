# 平均每次网络请求或响应字节数

本节为您介绍 OBProxy 的监控指标平均每次网络请求或响应字节数。

## client_request

### 指标介绍

客户端 request 字节数。

### 指标参数说明

|    **指标项**     |          **指标名称**           | **单位** |
|----------------|-----------------------------|--------|
| client_request | request_byte_client_request | byte   |

### 计算表达式

sum(rate(odp_request_byte{sessionType="client",transType="request",@LABELS}[@INTERVAL])) by (@GBLABELS)
