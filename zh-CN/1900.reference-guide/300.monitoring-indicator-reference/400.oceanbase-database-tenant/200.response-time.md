# 响应时间

本节为您介绍 OceanBase 租户的性能指标 **响应时间** 。响应时间指服务端处理单条 SQL 的耗时，具体指服务端从接收到 SQL 请求，到该 SQL 执行结束的时间 。您可指定统计范围，查询租户在单个 Zone 或 OBServer 节点的 **响应时间** 数据。
