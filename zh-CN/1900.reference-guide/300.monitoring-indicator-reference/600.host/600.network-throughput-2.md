# 网络吞吐率

本节为您介绍主机的性能指标 **网络吞吐率** 。

## receive

### 指标介绍

每秒接收数据量。

### 指标参数说明

| **指标项** | **指标名称** | **单位** |
|---------|----------|--------|
| receive | net_recv | MB     |

### 计算表达式

avg(rate(node_network_receive_bytes_total{device!="lo",@LABELS}[@INTERVAL])) by (@GBLABELS) / 1048576

## send

### 指标介绍

每秒发送数据量。

### 指标参数说明

| **指标项** | **指标名称** | **单位** |
|---------|----------|--------|
| send    | net_send | MB     |

### 计算表达式

avg(rate(node_network_transmit_bytes_total{device!="lo",@LABELS}[@INTERVAL])) by (@GBLABELS) / 1048576
