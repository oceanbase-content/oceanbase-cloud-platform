# 进程文件句柄数

本节为您介绍主机进程的监控指标 **进程文件句柄数** 。

## observer 进程

### 指标介绍

observer 进程文件句柄数。

### 指标参数说明

| **指标项** | **指标名称** | **单位** |
|---------|----------|--------|
| observer | observer_fd_count | 个     |

### 计算表达式

avg(process_fd_count{name="observer",@LABELS}) by (@GBLABELS)
