# CPU 使用率

本节为您介绍主机的性能指标 **CPU 使用率** 。

## 指标介绍

主机的 CPU 使用率。

## 指标参数说明

| **指标项** |  **指标名称**   | **单位** |
|---------|-------------|--------|
| CPU 使用率 | cpu_percent | %      |

## 计算表达式

100 * (1 - sum(rate(node_cpu_seconds_total{mode=~"idle|iowait|steal",@LABELS}[@INTERVAL])) by (@GBLABELS) / sum(cpu_count{@LABELS}) by (@GBLABELS))
