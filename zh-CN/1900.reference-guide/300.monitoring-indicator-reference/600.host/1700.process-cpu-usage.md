# 进程 CPU 使用率

本节为您介绍主机进程的监控指标 **进程 CPU 使用率** 。

## observer 进程

### 指标介绍

observer 进程 CPU 使用率。

### 指标参数说明

| **指标项** | **指标名称** | **单位** |
|---------|----------|--------|
| observer | observer_process_cpu_percent | %     |

### 计算表达式

avg(process_cpu_percent{name="observer",@LABELS}) by (@GBLABELS)
