# Inode 使用率

本节为您介绍主机 **Inode 使用率** 相关的监控指标。

## 指标介绍

服务器 inode 使用率。

## 指标参数说明

| **指标项** |    **指标名称**    | **单位** |
|---------|----------------|--------|
| file_inode_usage       | file_inode_usage | %     |

## 计算表达式

round(100 - 100 * sum(node_filesystem_files_free{@LABELS}) by (@GBLABELS) / sum(node_filesystem_files{@LABELS}) by (@GBLABELS))
