# OCP-Agent 进程协程数

本节为您介绍主机进程的监控指标 **OCP-Agent 进程协程数** 。

## monagent 进程

### 指标介绍

monagent 进程 GO 协程数量。

### 指标参数说明

| **指标项** | **指标名称** | **单位** |
|---------|----------|--------|
| monagent | monagent_goroutine_count | 个     |

### 计算表达式

max(go_goroutines{process="ocp_monagent",@LABELS}) by (@GBLABELS)
