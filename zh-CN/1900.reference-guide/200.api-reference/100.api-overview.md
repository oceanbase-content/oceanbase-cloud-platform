# 概述

为方便调用 API 管理您的 OceanBase 集群资源和开发自己的应用程序，OCP 公开符合 REST 规范的 HTTP API。本文档提供 OCP 开放 API 的描述、语法及示例。文档面向开发人员。在调用接口前，请确保已充分了解 OCP 平台的的相关概念。
