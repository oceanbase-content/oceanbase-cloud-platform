重启 OCP Agent
=================================

本页面为您介绍重启 OCP Agent 的操作步骤。

**操作步骤**
-----------------------------

1. 登录 OCP。

2. 在左侧导航栏中单击 **主机**。

3. 单击需要修改的主机后的 **重启 OCP Agent** 。

   ![1223](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/1423130461/p375344.png)

   系统弹出确认重启的操作提示。

4. 在弹出框中单击 **确定** ，重启 OCP Agent。
