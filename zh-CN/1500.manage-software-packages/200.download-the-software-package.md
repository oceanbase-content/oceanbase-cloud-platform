# 下载软件包

本页面为您介绍下载软件包的操作步骤。

## 前提条件

请确保当前登录 OCP 的用户具备如下权限：

* **集群只读**、**租户只读**、**OBProxy 只读**、**仲裁服务只读**、**Binlog 服务只读** 或 **主机只读** 任一资源权限。
* 系统管理 **软件包管理** 菜单权限。

## 操作步骤

1. 在左侧导航栏单击 **系统管理 > 软件包管理** ，进入软件包页面。

2. 单击对应软件包后的 **下载** ，可下载对应的软件包。
