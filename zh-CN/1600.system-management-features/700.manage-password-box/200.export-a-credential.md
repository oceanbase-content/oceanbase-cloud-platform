# 导出凭据

本节介绍导出凭据的操作步骤。

## 前提条件

当前登录用户须具有 `HOST_MANAGER`、`CLUSTER_MANAGER`、`TENANT_MANAGER` 或 `OBPROXY_MANAGER` 其中一个角色权限。

## 操作步骤

1. 登录 OCP。

2. 单击左导航栏 **系统管理** \> **凭据管理**，进入凭据列表页面。

3. 在 **OceanBase**、**OBProxy** 或 **主机** 页签中找到要导出的凭据对象。

4. 将光标悬停至该对象 **操作** 列的 **更多** ，在选择框中单击 **导出** 。

5. 在对话框中输入加密密码，并单击 **导出**。
