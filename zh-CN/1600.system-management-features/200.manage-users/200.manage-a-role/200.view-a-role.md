# 查看角色

本节介绍如何查看角色。

## 前提条件

当前登录用户为系统管理员（ADMIN）角色或组织管理员（ORG_ADMIN）角色。

## 操作步骤

1. 登录 OCP。

2. 在左侧导航栏中单击 **系统管理 > 用户管理** 。

3. 单击 **角色** ，进入 **角色** 页签。

   该页签的角色列表默认展示所有角色。角色过多时会分页展示，可通过列表下方的分页功能换页或调整每页展示角色数量。

4. 在搜索栏中填写角色名称关键字，并单击搜索按钮。

   角色列表中显示符合条件的角色。

5. 单击角色列表中具体某个角色的名称。

   进入该角色的 **角色详情** 页，该页中可查看角色的详细信息。
