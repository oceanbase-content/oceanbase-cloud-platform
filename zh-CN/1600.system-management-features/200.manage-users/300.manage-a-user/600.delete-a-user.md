# 删除用户

本节介绍如何删除用户。

## 前提条件

* 请确保当前登录 OCP 的用户为 **ADMIN** 角色或 **ORG_ADMIN** 角色。

* 不支持删除 admin 用户。

## 操作步骤

1. 登录 OCP。

2. 在左侧导航栏中单击 **系统管理 > 用户管理** 。

3. 在 **用户** 页签单击用户列表 **操作** 列的 **更多 > 删除**。

   系统提示 "确定要删除用户 xx 吗？"。

4. 单击 **删除** 。

  <main id="notice" type='notice'>
    <h4>注意</h4>
    <p>用户删除后不可恢复，请谨慎操作。</p>
  </main>
