# 忽略连接

本节介绍如何忽略连接。当其他 OCP 用户向您分享连接凭证时，您可选择是否忽略该连接。

## 操作步骤

1. 在 OCP 系统界面的右上角，单击当前登录用户名，选择 **密码箱**。
   进入 **密码箱** 页面。
   出于安全考虑，页面上的密码均显示为 *******。

2. 单击右上角 **待接受的连接** 按钮。

3. 在 **OceanBase 集群** 或 **主机** 页签中选择要忽略的连接对象。

4. 单击该对象 **操作** 列的 **忽略** 按钮。
   此外，在 **OceanBase 集群** 页签中，您可单击 **全部忽略** 按钮，忽略所有连接。

5. 在弹出的对话框中单击 **忽略**。
