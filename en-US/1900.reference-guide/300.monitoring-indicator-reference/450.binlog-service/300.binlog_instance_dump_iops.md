# Network Traffic

This topic describes the subscription connection monitoring metric **Network Traffic** of a binlog instance. You can specify a statistical range to query the network traffic data of a binlog instance.

## Definition

The transmit traffic of a binlog instance.

## Description

| **Metric** |   **Metric field name**    | **Unit** |
|---------|---------------|--------|
| all     | binlog_instance_dump_iops | Byte      |

## Calculation expression

sum(binlog_instance_dump_iops{@LABELS}) by (@GBLABELS)
