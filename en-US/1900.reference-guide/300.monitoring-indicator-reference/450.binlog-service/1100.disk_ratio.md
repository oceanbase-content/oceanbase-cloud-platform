# disk ratio

This topic describes the resource monitoring metric **disk ratio** of a binlog instance.

## Definition

The disk usage of a binlog instance in percentage.

## Description

| **Metric** |   **Metric field name**    | **Unit** |
|---------|---------------|--------|
| disk_ratio     | binlog_service_disk_usage | %     |
