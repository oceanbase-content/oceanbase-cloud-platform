# Collect the last-one-day records of major compaction events scheduled by RootService

## Description

You can query the historical events related to the `daily_merge` module at the cluster level.

## Statement

OceanBase Database V3.2.0 and later:

```sql
SELECT * FROM __all_rootservice_event_history
WHERE module = 'daily_merge' AND gmt_create >= NOW() - INTERVAL 1 DAY
ORDER BY gmt_create DESC;
```
